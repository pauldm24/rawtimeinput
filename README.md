# React Raw Time Input

Project dependencies

* react
* moment

### Usage
Install the package
Install the package.
```bash
yarn add rawtimeinput # npm install rawtimeinput --save
```

### Demo

```bash
git clone git@github.com:pauldm24/rawtimeinput.git
cd rawtimeinput
yarn # npm install

```
